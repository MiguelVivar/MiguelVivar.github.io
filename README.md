## 🎨 Tema Claro y Oscuro

Mi portafolio incluye soporte para **modo claro y oscuro**, brindando una experiencia personalizada para cada usuario.  

- **Modo Claro**: Ideal para entornos luminosos, con colores vibrantes y alta visibilidad.  
- **Modo Oscuro**: Diseñado para reducir la fatiga visual en entornos oscuros, con tonos suaves y elegantes.  

El tema se ajusta automáticamente según las preferencias del sistema operativo del usuario o se puede cambiar manualmente mediante un interruptor en el sitio.

## 🖥️ Instalación Local
Sigue estos pasos para ejecutar el proyecto en tu máquina local:
1. Clona este repositorio:
```{.line-numbers}
    git clone https://github.com/MiguelVivar/miguelvivar.github.io.git
```
2. Abre la carpeta:
```{.line-numbers}
    git clone cd miguelvivar.github.io
```
3. Instala las dependencias:
```{.line-numbers}
    npm install
```
4. Inicia el servidor de desarrollo:
```{.line-numbers}
    npm run dev
```
5. Abren en el navegador en http://localhost:5173 para ver el sitio en acción.
